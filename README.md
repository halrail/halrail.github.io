# 📚 時間割通知アプリ（PWA対応）

**授業の 5 分前に通知してくれる Web アプリ**です。  
平日は 1〜7 限、土曜は必要な週だけ 1〜4 限を設定できます。  
科目・持ち物・開始時刻を入力すると、毎週自動で通知されます。

- ✔ iPhone / Android / PC 対応  
- ✔ オフラインでも動作  
- ✔ PWA（ホーム画面追加）対応  
- ✔ JSON バックアップ / 復元  
- ✔ 土曜授業は「今週だけ ON」にできる  
- ✔ サーバー不要（GitHub Pages で動く）
